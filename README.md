# Walmart-Sales-Analysis-Power-Bi-
This project focuses on analyzing Walmart sales data using Power BI to uncover key business insights, trends, and performance metrics. The goal is to transform raw retail data into meaningful visualizations that support data-driven decision-making.

**🚀 Project Overview**

The dataset was preprocessed and cleaned using Python (Pandas), followed by data modeling and visualization in Power BI. The dashboard provides a comprehensive view of sales performance across different stores, time periods, and categories.

**🔧 Tools & Technologies Used**
Python (Pandas) – Data cleaning and preprocessing
Power BI – Data visualization and dashboard creation
Excel/CSV – Data source

**📌 Key Features**
📈 Sales Trend Analysis (Monthly & Yearly)
🏬 Store-wise Performance Comparison

**🎯 Key Performance Indicators (KPIs):**
Total Sales
Average Sales
Total Stores
📊 Sales Category Segmentation (High / Medium / Low)
📅 Holiday vs Non-Holiday Sales Analysis
🎛️ Interactive Filters (Year, Month, Store)

**📊 Dashboard Insights**
Identified top-performing stores contributing maximum revenue
Observed seasonal trends and holiday impact on sales
Segmented sales into categories for better business understanding
Provided actionable insights for improving store performance

**🧹 Data Preprocessing Steps**
Removed missing and duplicate values
Converted data types (Date, Numeric fields)
Created new columns like:
Sales Category
Aggregated metrics for analysis

**📌 Conclusion**

This project demonstrates how raw retail data can be transformed into meaningful insights using data analytics tools. It highlights the importance of visualization in identifying trends, improving decision-making, and driving business growth.
